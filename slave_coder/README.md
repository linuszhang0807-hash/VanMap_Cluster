# 展示平台 (Coder Module)

本模块负责情报的视觉化呈现与用户交互。

## 核心职责
- **UI 渲染**: 使用奶白风格渲染餐厅列表与地图标记。
- **高级筛选系统**: 实现两级分类联动（类目->子类目）及地区交集筛选。
- **交互逻辑**: 
    - 实现餐厅评分降序排列。
    - 实现视频链接首字母 A-Z 排序。
    - 鼠标悬停（Hover）弹出预览窗口。

## 快速部署
1. **启动服务**: 在 `slave_coder/` 根目录下执行：
   `python -m http.server 8080`
2. **访问**: 在浏览器输入 `http://localhost:8080`。

## 交互逻辑说明
- **交集逻辑**: 筛选器（分类 + 地区）采用 AND 逻辑，确保显示结果精确。
- **数据源**: 读取 `src/shared_data/restaurants.json`。
